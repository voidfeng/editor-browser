# 📁 文件资源管理器功能

## 概述

我已经成功为Editor Browser添加了一个类似VSCode的左侧树形文件资源管理器组件，大大提升了代码编辑体验。

## 🆕 新增功能

### 1. 文件资源管理器 (FileExplorer.vue)

- **树形文件结构**：以层级方式显示项目文件和文件夹
- **文件夹操作**：支持展开/折叠文件夹
- **文件信息**：显示文件大小和修改时间
- **刷新功能**：一键刷新文件列表
- **折叠所有**：快速折叠所有展开的文件夹

### 2. 文件树节点 (FileTreeNode.vue)

- **智能图标**：根据文件类型显示不同图标
- **交互反馈**：悬停和选中状态的视觉反馈
- **右键菜单**：支持右键菜单（可扩展）
- **文件大小显示**：实时显示文件大小信息

### 3. 增强的浮动编辑器 (FloatingEditor.vue)

- **侧边栏集成**：左侧集成文件资源管理器
- **多标签页**：支持同时打开多个文件
- **文件切换**：便捷的文件标签页切换
- **智能语言识别**：根据文件扩展名自动设置语法高亮
- **侧边栏切换**：可隐藏/显示左侧边栏

## 🎨 界面特性

### VSCode风格设计

- 采用GitHub风格的配色方案
- 清晰的层级结构显示
- 现代化的图标系统
- 流畅的动画过渡效果

### 响应式布局

- 自适应窗口大小
- 可拖拽的编辑器窗口
- 灵活的侧边栏宽度
- 优化的滚动条样式

## 🚀 使用方法

### 基本操作

1. **打开编辑器**：点击浮动按钮 📝
2. **浏览文件**：在左侧资源管理器中导航
3. **打开文件**：点击文件名在编辑器中打开
4. **切换文件**：使用顶部标签页切换已打开的文件
5. **关闭文件**：点击标签页上的 ✕ 按钮

### 高级功能

- **侧边栏切换**：点击标题栏的 📁 按钮
- **搜索功能**：切换到搜索标签页（待完善）
- **主题切换**：🌙/☀️ 按钮切换明暗主题
- **代码执行**：▶️ 按钮运行JavaScript代码

## 🔧 技术实现

### 组件架构

```
FloatingEditor.vue (主编辑器)
├── FileExplorer.vue (文件资源管理器)
│   └── FileTreeNode.vue (树节点组件)
└── CodeEditor.vue (代码编辑器)
```

### 核心特性

- **Vue 3 Composition API**：现代化的组件开发

- **响应式数据**：实时的文件状态管理
- **事件驱动**：组件间的高效通信
