# Editor Browser 使用说明

## 功能特性

这个 Chrome 扩展会在任何网页中注入一个浮动的代码编辑器，让你可以随时随地编写和测试代码。

## 主要功能

### 🚀 浮动编辑器
- 在任何网页右上角显示一个浮动按钮（📝）
- 点击按钮打开/关闭编辑器面板
- 支持拖拽移动编辑器位置

### 💻 代码编辑
- 基于 CodeMirror 6 的强大编辑器
- 语法高亮支持：JavaScript、HTML、CSS、JSON
- 行号显示、代码折叠、自动补全
- 括号匹配、历史记录（撤销/重做）

### 🎨 主题切换
- 支持亮色/暗色主题
- 点击 🌙/☀️ 按钮切换主题

### ▶️ 代码执行
- JavaScript 代码可以直接运行
- 捕获 console.log 输出
- 显示执行结果或错误信息

### ⌨️ 快捷键
- `Ctrl/Cmd + E`: 切换编辑器显示/隐藏
- `Ctrl/Cmd + Enter`: 运行当前代码

### 🔧 界面控制
- 最小化按钮：收起编辑器内容，只保留标题栏
- 关闭按钮：完全隐藏编辑器
- 拖拽标题栏：移动编辑器位置

## 安装方法

1. 运行 `npm run build` 构建扩展
2. 打开 Chrome 浏览器，进入 `chrome://extensions/`
3. 开启"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择项目的 `dist` 文件夹

## 使用场景

- 📚 学习编程时快速测试代码片段
- 🐛 调试网页时编写测试脚本
- 💡 灵感来临时随时记录代码想法
- 🔧 在任何网站上编写辅助脚本
- 📝 代码面试或技术讨论时的临时编辑器

## 技术栈

- Vue 3 + TypeScript
- CodeMirror 6
- Chrome Extension Manifest V3
- Vite + CRXJS

现在你可以在任何网页上享受强大的代码编辑体验！🎉
