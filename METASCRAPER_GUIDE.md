# Metascraper 页面分析功能使用指南

## 🎯 功能概述

Editor Browser 扩展现在集成了 metascraper 库，可以将任何网页转换为结构化的 JSON 数据。这个功能让你能够：

- 提取页面元数据（标题、描述、作者等）
- 分析页面结构（标题、链接、图片、表单）
- 统计页面内容（字数、预估阅读时间）
- 生成完整的页面数据 JSON

## 🚀 如何使用

### 1. 打开编辑器

- 在任何网页上找到右上角的 📝 浮动按钮
- 点击按钮打开编辑器面板

### 2. 分析页面

- 在编辑器工具栏中点击 **🔍 分析页面** 按钮
- 等待分析完成（通常几秒钟
